# Startup Validator

This version is frontend-only. It runs entirely in the browser with no API key, no backend, and no installation.

## How to Run

Open [frontend/index.html](C:/Users/ashas/OneDrive/Documents/Recommendation%20system/startup-validator/frontend/index.html) in any browser.


## What It Does

- Accepts a startup idea
- Generates a rule-based SWOT analysis
- Estimates market potential
- Suggests likely competitors
- Gives a viability score and practical next steps
